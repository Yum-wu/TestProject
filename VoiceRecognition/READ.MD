# 🎙️ AI 语音识别应用

基于浏览器录音与智谱 AI 语音识别 API 的轻量级 Web 应用，支持录音、识别、历史记录管理，全程通过自然语言与 AI 对话开发完成。

---

## ✨ 功能特性

- ✅ 浏览器端录音（MediaRecorder API）
- ✅ 调用智谱 AI 语音识别 API，中文识别
- ✅ 录音按钮脉冲动画，状态清晰
- ✅ 识别结果可编辑、一键复制
- ✅ 加载动画与友好错误提示
- ✅ 录音时长限制（最长 60 秒）+ 实时倒计时
- ✅ 历史记录本地存储（LocalStorage），支持查看与删除

---

## 🛠️ 技术栈

- 前端：HTML5 / CSS3 / JavaScript (ES6+)
- 录音：MediaRecorder API
- 语音识别：智谱 AI 语音识别 API (HTTP)
- 存储：LocalStorage

## 📋 开发技巧

### 1. 麦克风权限处理
- 浏览器首次调用 `getUserMedia` 时会弹出权限请求。
- 若用户拒绝，需提供友好提示并显示重新请求权限的按钮。
- 提示文案示例：
  “需要麦克风权限才能录音，请在浏览器设置中允许访问麦克风。”

### 2. 音频格式
- MediaRecorder 默认输出格式为 `webm`（Chrome、Edge 等），智谱 AI 等主流 API 均支持该格式。
- 如需转换格式（如 wav），可与 AI 协作完成前端转换。

### 3. 录音时长限制
- 建议设置最长录音时长（如 60 秒），避免文件过大、等待过长。
- 实时显示录音时长，到达限定时间自动停止。
- 使用 `setTimeout` 或 `setInterval` 实现倒计时提示。

## ⚙️ 使用说明

1. 克隆或下载本项目，在浏览器中打开 `index.html`。
2. 首次点击麦克风按钮时，授予麦克风权限。
3. 点击录音按钮开始讲话，再次点击停止录音。
4. 点击“识别”按钮上传音频至智谱 API，等待识别结果。
5. 结果可编辑、复制，并自动存入历史记录。
6. 底部可查看/删除历史记录。

> 💡 请确保在 `app.js` 中配置有效的智谱 AI API Key。
