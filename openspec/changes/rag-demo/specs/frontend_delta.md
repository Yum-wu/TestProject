### Project: `Chatbot/rag-ui/` — RAG 知识库问答前端

独立的 React + Vite 前端项目，提供 RAG 知识库问答的交互界面。

#### Scenario: 知识问答
- GIVEN 用户打开 RAG 问答页面
- WHEN 用户在输入框中输入问题并点击"发送"
- THEN 调用 `POST /api/rag/query` 接口
- AND 页面展示 LLM 生成的回答
- AND 回答下方标注引用的博文来源

#### Scenario: 来源查看
- GIVEN 页面展示了包含来源的 RAG 回答
- WHEN 用户点击引用来源卡片
- THEN 展开显示该来源的原文切片内容
- AND 提供"查看原文"链接跳转到博文
