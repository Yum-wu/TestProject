# Hello World!

Welcome to my personal blog 🎉

This is my first post. This blog is built with **React** + **Vite**, all articles are managed as Markdown files, and deployed via **Vercel** — completely free.

## Why Write a Blog?

I've always believed that writing is the best way to learn. Putting knowledge into your own words not only deepens understanding but also helps others along the way.

> Teaching is the best way to learn.

## Tech Stack

For the frontend framework, I chose:

- **React 19** — The latest React version
- **Vite 8** — Blazing fast build tool
- **TypeScript** — Type safety
- **Tailwind CSS 4** — Atomic CSS
- **React Router 7** — Frontend routing
- **React Markdown** — Markdown rendering
- **Giscus** — GitHub Discussions based comment system

## Article Management

Articles are stored as Markdown files in the repository, version-controlled through Git. The benefits of this approach:

1. **Simple** — No database, no backend needed
2. **Version Control** — Naturally has full history of all articles
3. **Free Deployment** — Completely zero cost
4. **Familiar Editing Experience** — Write with any Markdown editor
