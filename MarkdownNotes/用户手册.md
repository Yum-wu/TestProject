# Markdown 笔记应用 - 用户指南

欢迎使用 Markdown 笔记应用！这是一份专门为非编程用户编写的详细使用手册。

---

## 一、快速开始

### 1.1 启动应用

1. 双击桌面上的 `MarkdownNotes` 文件夹
2. 打开终端，输入以下命令：
   ```
   npm run dev
   ```
3. 当看到提示 `Local: http://localhost:5173/` 时
4. 打开浏览器，访问 `http://localhost:5173/`

### 1.2 界面概览

应用界面分为三个区域：

- **左侧区域**：笔记列表（包含搜索功能）
- **中间区域**：编辑器（编写 Markdown 内容）
- **右侧区域**：实时预览（显示渲染后的效果）

---

## 二、核心功能

### 2.1 创建笔记

**方法一**：点击左下角绿色的 `+ 新建笔记` 按钮

**方法二**：使用快捷键 `Ctrl + N`（Windows）或 `Cmd + N`（Mac）

创建后，笔记会自动出现在左侧列表中。

### 2.2 编辑笔记

1. 在左侧列表中点击要编辑的笔记
2. 中间编辑器区域会显示该笔记内容
3. 开始输入文字，系统会自动提取第一行作为笔记标题
4. 右侧会实时显示渲染后的效果

**提示**：编辑器会自动聚焦光标，打开后可以直接输入。

### 2.3 搜索笔记

1. 在左侧搜索框中输入关键词
2. 列表会实时过滤匹配的笔记
3. 搜索不区分大小写
4. 清空搜索框可以恢复显示全部笔记

### 2.4 删除笔记

**方法一**：在左侧笔记列表中，鼠标悬停在笔记上，点击右侧出现的删除图标

**方法二**：在右侧预览区域，点击顶部的 `删除` 按钮

删除前会弹出确认对话框，防止误删。

## 三、Markdown 语法基础

### 3.1 标题

使用 `#` 号表示标题级别：

```
# 一级标题（最大）
## 二级标题
### 三级标题
#### 四级标题
```

### 3.2 文字格式

- **加粗**：`**加粗文字**` 或 `__加粗文字__`
- *斜体*：`*斜体文字*` 或 `_斜体文字_`
- ~~删除线~~：`~~删除文字~~`
- `行内代码`：`` `代码内容` ``

### 3.3 列表

**无序列表**（使用 `-`、`*` 或 `+`）：
```
- 第一项
- 第二项
  - 子项（缩进表示层级）
```

**有序列表**（使用数字）：
```
1. 第一项
2. 第二项
3. 第三项
```

**任务列表**：
```
- [ ] 未完成的任务
- [x] 已完成的任务
```

### 3.4 引用

使用 `>` 符号：
```
> 这是一段引用文字
> 可以换行
```

### 3.5 代码块

使用三个反引号（```）包裹代码：

````
```javascript
function hello() {
  console.log("Hello, World!");
}
```
````

支持的编程语言包括：javascript, typescript, python, java, c++, go, rust, html, css 等。

### 3.6 表格

```
| 表头 1 | 表头 2 | 表头 3 |
| ------ | ------ | ------ |
| 内容 1 | 内容 2 | 内容 3 |
| 内容 4 | 内容 5 | 内容 6 |
```

### 3.7 分割线

使用三个或以上的 `-`、`*` 或 `_`：
```
---
```

### 3.8 链接和图片

**链接**：
```
[链接文字](https://example.com)
```

**图片**：
```
![图片描述](图片地址)
```

## 四、导出功能

### 4.1 导出为 Markdown 文件

1. 在右侧预览区域，点击顶部的 `导出 MD` 按钮
2. 浏览器会下载一个 `.md` 文件
3. 文件名自动使用笔记标题

### 4.2 导出为 PDF

1. 在右侧预览区域，点击顶部的 `导出 PDF` 按钮
2. 浏览器会打开打印对话框
3. 选择 `另存为 PDF` 选项
4. 点击 `保存` 完成导出

---

## 五、数据管理

### 5.1 自动保存

- 笔记内容会自动保存到浏览器本地存储
- 修改后约 0.5 秒自动保存
- 底部状态栏会显示 `● 已保存` 或 `● 未保存` 状态

### 5.2 数据持久化

- 所有数据保存在浏览器中
- 刷新页面数据不会丢失
- 关闭浏览器后再打开，数据依然存在

### 5.3 注意事项

- 数据仅保存在当前浏览器中
- 清除浏览器数据会导致笔记丢失
- 建议定期使用 `导出 MD` 功能备份重要笔记
- 不支持多设备同步

## 六、常见问题

### Q1：为什么右侧预览不更新？

**答**：检查中间编辑器是否已正确输入内容。Markdown 语法需要遵循规范，错误的语法可能导致渲染失败。

### Q2：代码高亮不显示？

**答**：在代码块的第一行反引号后指定语言名称，例如：
````
```javascript
// 代码内容
```
````

### Q3：如何恢复误删的笔记？

**答**：当前版本不支持回收站功能。建议：
- 删除前仔细确认
- 定期导出重要笔记备份
- 使用浏览器历史记录恢复（如果尚未关闭）

### Q4：笔记太多，加载很慢？

**答**：
- 使用搜索功能快速定位
- 定期清理不需要的笔记
- 导出备份后删除旧笔记

### Q5：可以在手机上使用吗？

**答**：当前版本针对桌面端优化，移动端显示可能不够理想。

## 八、技术支持

### 8.1 项目信息

- 技术栈：React + TypeScript + Vite
- Markdown 解析：react-markdown
- 代码高亮：react-syntax-highlighter
- 样式：Tailwind CSS
- 数据存储：LocalStorage

### 8.2 常见问题排查

1. **应用无法启动**
   - 检查 Node.js 是否安装（需要 v18 或更高版本）
   - 检查依赖是否正确安装（运行 `npm install`）

2. **浏览器控制台报错**
   - 打开浏览器开发者工具（F12）
   - 查看 Console 标签的错误信息
   - 截图保存以便排查

3. **样式显示异常**
   - 尝试清除浏览器缓存
   - 检查 Tailwind CSS 是否正确配置
