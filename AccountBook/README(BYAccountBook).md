# 个人记账本项目

## 项目简介

这是一个基于Python面向对象编程的个人记账本应用，用于管理个人财务收支记录。项目采用模块化设计，具有良好的可扩展性和可维护性。

## 功能需求

### 核心功能
1. **交易管理**
   - 添加收入/支出记录
   - 删除交易记录
   - 修改交易信息
   - 查询交易记录

2. **分类管理**
   - 自定义收入/支出类别
   - 类别层级管理

3. **统计分析**
   - 月度收支统计
   - 类别支出分析
   - 余额计算

4. **数据持久化**
   - 本地文件存储
   - 数据导入导出

## 项目架构设计

### 面向对象类结构

#### 1. Transaction类（交易记录）
**职责**：表示单笔交易记录
- **属性**：
  - `transaction_id`：交易唯一标识
  - `date`：交易日期
  - `type`：交易类型（收入/支出）
  - `amount`：交易金额
  - `category`：交易类别
  - `description`：交易描述
- **方法**：
  - `__init__()`：初始化交易记录
  - `__str__()`：返回交易信息的字符串表示
  - `validate()`：验证交易数据的合法性

#### 2. Category类（交易类别）
**职责**：管理交易类别信息
- **属性**：
  - `name`：类别名称
  - `type`：类别类型（收入/支出）
  - `parent`：父类别（支持层级结构）
- **方法**：
  - `__init__()`：初始化类别
  - `is_valid()`：验证类别合法性

#### 3. AccountBook类（记账本核心）
**职责**：管理所有交易记录和提供核心功能
- **属性**：
  - `transactions`：交易记录列表
  - `categories`：类别列表
  - `balance`：当前余额
- **方法**：
  - `add_transaction()`：添加新交易
  - `remove_transaction()`：删除交易
  - `get_transactions()`：查询交易记录
  - `calculate_balance()`：计算当前余额
  - `generate_report()`：生成统计报表
  - `save_to_file()`：保存数据到文件
  - `load_from_file()`：从文件加载数据

#### 4. AccountBookUI类（用户界面）
**职责**：提供用户交互界面
- **属性**：
  - `account_book`：AccountBook实例
- **方法**：
  - `show_menu()`：显示主菜单
  - `handle_input()`：处理用户输入
  - `display_transactions()`：显示交易记录
  - `display_report()`：显示统计报表

## 技术栈

- **编程语言**：Python 3.x
- **数据存储**：JSON文件格式
- **开发工具**：标准Python库（datetime, json, os等）

## 项目特点

1. **面向对象设计**：采用清晰的类层次结构，符合单一职责原则
2. **模块化架构**：各功能模块独立，便于维护和扩展
3. **数据验证**：完善的输入验证和错误处理机制
4. **可扩展性**：支持功能扩展和自定义类别

## 使用方法

1. 运行主程序：`python AccountBook.py`
2. 按照菜单提示进行操作
3. 支持添加、查询、删除交易记录
4. 可生成收支统计报表
