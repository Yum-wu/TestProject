## 产品简介

Todo Manager 是一款简洁高效的个人待办事项管理工具，帮助您有序管理日常任务，提升工作和生活效率。

### 您可以用它做什么？

- **记录任务** - 随时记录需要完成的事情
- **跟踪进度** - 标记已完成和未完成的任务
- **分类整理** - 按工作、生活、学习等分类管理
- **设置期限** - 为任务设置截止日期，避免遗忘
- **快速查找** - 通过搜索和筛选快速找到任务

### 产品特点

- **简单易用** - 打开浏览器即可使用，无需注册登录
- **数据安全** - 数据保存在您的设备本地，不会上传到任何服务器
- **多端可用** - 电脑和手机浏览器都可以使用

## 添加任务

### 快速添加

最简单的添加方式：

1. 在输入框中输入任务内容
2. 按键盘上的 `Enter` 键，或点击「添加」按钮

**示例**：
```
输入: "明天下午3点开会" → 按 Enter → 任务已添加到列表
```

### 添加详细信息

如果需要为任务添加更多说明，可以展开详细表单：

1. 点击输入框，会展开更多信息选项
2. 填写以下内容（可选）：

| 字段 | 说明 | 是否必填 |
| :--- | :--- | :--- |
| 任务标题 | 任务的简要描述 | 必填 |
| 任务描述 | 更详细的说明 | 可选 |
| 截止日期 | 任务需要完成的日期 | 可选 |
| 分类标签 | 为任务分配分类 | 可选 |

**示例**：
```
任务标题: 完成项目报告
任务描述: 需要汇总各部门数据，包含图表分析
截止日期: 2026年5月1日
分类标签: 工作
```

### 小技巧

- 输入框添加任务后会自动清空，可以连续添加多个任务
- 分类标签输入时会显示已有分类供选择

## 完成任务

当您完成某项任务后，可以将其标记为已完成：

1. 找到要完成的任务
2. 点击任务左侧的圆形复选框

**效果**：
```
○ 未完成任务 → 点击 → ◉ 已完成任务（灰色显示，带删除线）
```

**再次切换**：
- 如果误操作，再次点击可以恢复为未完成状态

---

## 编辑任务

如果需要修改任务内容，可以：

### 方法一：双击编辑

1. 双击任务标题，进入编辑模式
2. 修改任务标题或描述
3. 点击「保存」按钮

### 方法二：点击编辑按钮

1. 点击任务右侧的铅笔图标
2. 修改任务信息
3. 点击「保存」按钮

**编辑模式界面**：
```
┌──────────────────────────────────┐
│ [修改后的任务标题.......]        │
│ [修改后的描述.........]          │
│                    [取消] [保存]  │
└──────────────────────────────────┘
```

- **保存** - 确认修改
- **取消** - 放弃修改，恢复原内容

---

## 删除任务

如果某个任务不再需要，可以删除它：

1. 找到要删除的任务
2. 点击任务右侧的垃圾桶图标
3. 在弹出的确认对话框中点击「确定」

> **注意**：删除后无法恢复，请谨慎操作！

## 搜索任务

当任务较多时，可以使用搜索功能快速查找：

### 使用方法

1. 在搜索框中输入关键词
2. 列表会实时显示匹配的任务

### 搜索示例

```
搜索: "报告"

结果:
  ◉ 完成项目报告（已完成）
  ○ 准备季度报告
```

### 搜索特性

- **即时搜索** - 输入时就会更新结果
- **不区分大小写** - 搜索"Report"和"report"效果相同
- **全面搜索** - 会同时搜索任务标题和描述

## 筛选任务

### 按完成状态筛选

界面上方有三个状态按钮：

| 按钮 | 显示内容 | 适用场景 |
| :--- | :--- | :--- |
| **全部** | 所有任务 | 查看所有任务 |
| **进行中** | 未完成的任务 | 查看待办事项 |
| **已完成** | 已完成的任务 | 回顾已完成工作 |

### 筛选示例

**全部视图**（显示所有任务）
```
○ 购买生活用品
○ 准备会议材料
◉ 完成月度报告
```

**进行中视图**（只显示未完成的）
```
○ 购买生活用品
○ 准备会议材料
```

**已完成视图**（只显示已完成的）
```
◉ 完成月度报告
```

### 组合筛选

您可以同时使用状态筛选和分类筛选，精确找到目标任务。

**示例**：选择"进行中" + 选择"工作"分类
→ 显示所有工作中未完成的任务

## 分类管理

### 什么是分类？

分类帮助您更好地组织任务。例如：

- **工作** - 工作任务相关
- **生活** - 日常生活琐事
- **学习** - 学习和自我提升

### 创建分类

添加任务时，输入新的分类名称即可自动创建：

1. 展开任务详细表单
2. 在「分类标签」框中输入新的分类名
3. 添加任务后，新分类自动创建

**示例**：
```
分类标签输入: "健身" → 添加任务 → "健身" 分类已出现
```

### 使用已有分类

输入时可以选择已有分类：

1. 点击分类输入框
2. 从下拉列表中选择一个分类
3. 或者直接输入完整分类名

### 按分类查看

点击分类标签，只显示该分类的任务：

```
分类: [全部] [工作] [生活] [学习]
         ↑      ↑      ↑      ↑
      所有任务  工作   生活   学习
```

### 删除分类

点击分类标签右侧的 `×` 号，可以删除该分类。

> **注意**：删除分类不会删除任务，只是移除分类标签。

## 设置截止日期

### 如何设置日期

1. 展开任务详细表单
2. 点击「截止日期」输入框
3. 选择日期或直接输入日期

### 日期提醒颜色

系统会用不同颜色提醒您任务的时间状态：

| 颜色 | 含义 | 示例 |
| :--- | :--- | :--- |
| **蓝色** | 截止日期未到 | 📅 5月1日 |
| **黄色** | 今天是截止日期 | 📅 4月28日（今天）|
| **红色** | 已过期且未完成 | 📅 4月25日（已过期）|

### 过期提醒

如果任务过期仍未完成，任务左侧会显示红色边框，提醒您优先处理。

## 数据安全

### 数据存储位置

您的所有任务数据都保存在浏览器的本地存储中：

- 不需要注册账户
- 不需要联网使用
- 数据保存在您的设备本地

### 自动保存

- 每次添加、编辑、删除任务都会自动保存
- 刷新页面或关闭浏览器后，数据不会丢失

### 注意事项

⚠️ **以下情况会导致数据丢失**：

- 清除浏览器数据（如清除浏览记录）
- 更换浏览器（如从 Chrome 换到 Edge）
- 使用隐私/无痕模式（关闭后数据会清除）

### 建议

- 始终使用同一个浏览器访问
- 不要使用隐私/无痕模式
- 如果经常使用，可以定期记录重要任务

## 常见问题

### Q: 我需要注册账户吗？

A: 不需要。打开浏览器即可直接使用，无需注册或登录。

### Q: 数据会丢失吗？

A: 正常使用不会丢失。但如果清除浏览器数据或更换浏览器，数据会丢失。

### Q: 可以在手机和电脑上同步使用吗？

A: 当前版本不支持同步。数据保存在当前浏览器中，不同设备看到的是不同的数据。

### Q: 可以添加多少个任务？

A: 可以添加数百个任务，完全满足个人日常使用需求。

### Q: 为什么删除任务前要弹窗确认？

A: 为了防止误操作。删除的任务无法恢复，确认提示可以避免意外删除。

### Q: 分类可以重命名吗？

A: 暂时不支持直接修改分类名。您可以创建新分类，将任务移动过去后删除旧分类。

### Q: 任务可以排序吗？

A: 当前按添加时间显示，新任务显示在列表底部。后续版本可能会加入自定义排序功能。

### Q: 任务可以设置提醒吗？

A: 当前版本支持截止日期显示和过期提醒，暂不支持定时提醒通知。

### Q: 如何备份我的任务数据？

A: 当前版本暂不支持导出功能，后续版本会加入数据导入/导出功能。

### Q: 支持哪些浏览器？

A: 支持主流的浏览器，包括：

- Google Chrome（推荐）
- Microsoft Edge
- Mozilla Firefox
- Apple Safari
