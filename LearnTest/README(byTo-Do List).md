# 待办事项列表项目

## 项目简介

这是一个使用 Python 面向对象编程实现的待办事项列表应用，用于管理个人任务。

## 功能需求

### 核心功能
1. **任务管理**
   - 添加新任务（标题和描述）
   - 标记任务为已完成/未完成
   - 移除任务
   - 查看任务列表

2. **任务分类**
   - 查看未完成任务
   - 查看已完成任务

3. **用户交互**
   - 命令行界面
   - 友好的提示信息
   - 错误处理

## 实现步骤

### 步骤 1：设计类结构

1. **Task 类**（单个任务）
   - 属性：标题、描述、完成状态
   - 方法：初始化、标记完成、标记未完成、字符串表示

2. **TodoList 类**（任务管理）
   - 属性：任务列表
   - 方法：添加任务、移除任务、标记任务完成、获取未完成任务、获取已完成任务、显示任务列表

### 步骤 2：实现核心功能

1. **Task 类实现**
   - 构造函数：初始化任务属性
   - mark_completed()：设置完成状态为 True
   - mark_pending()：设置完成状态为 False
   - __str__()：返回任务的字符串表示

2. **TodoList 类实现**
   - 构造函数：初始化空任务列表
   - add_task()：创建并添加新任务
   - remove_task()：根据索引移除任务
   - mark_task_completed()：标记任务为已完成
   - get_pending_tasks()：返回未完成任务列表
   - get_completed_tasks()：返回已完成任务列表
   - display()：显示所有任务

### 步骤 3：添加示例用法

1. 创建待办事项列表实例
2. 添加几个测试任务
3. 显示初始任务列表
4. 标记一个任务为已完成
5. 显示更新后的任务列表
6. 移除一个任务
7. 显示最终任务列表
8. 查看未完成任务

## 使用说明

1. **运行程序**
   ```bash
   python "To-Do List.py"
   ```

2. **示例输出**
   ```

   1. [✗] 完成Python作业
   2. [✗] 去超市购物
   3. [✗] 健身
   ===================

   === 待办事项列表 ===
   1. [✓] 完成Python作业
   2. [✗] 去超市购物
   3. [✗] 健身
   ===================

   === 待办事项列表 ===
   1. [✓] 完成Python作业
   2. [✗] 健身

   未完成任务：
   - [✗] 健身
   ```

## 扩展建议

1. **持久化存储**：将任务保存到文件或数据库
2. **优先级管理**：为任务添加优先级
3. **截止日期**：为任务添加截止日期
4. **图形界面**：使用 Tkinter 或 PyQt 创建 GUI
5. **搜索功能**：根据关键词搜索任务
