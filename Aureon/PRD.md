# AI 聊天助手 — Agent 升级 PRD

## 核心功能

1. **Agent 智能路由**：用户问题由 LangChain Agent 自动判断，选择直接对话或调用工具
2. **工具调用（Tool Calling）**：
   - **计算器**：数学表达式自动求值（四则运算、幂、三角函数）
   - **联网搜索**：Tavily API 实时检索最新信息（可选，需 API Key）
   - **外存读取**：Agent 可按需读取已归档的完整工具输出
3. **流式输出**：AI 回答逐字显示，工具调用过程实时告知用户
4. **四层记忆系统**：
   - L0 原始对话存储（SQLite，上限 500 条/会话）
   - L1 原子事实提取（用户偏好、关键信息结构化入库）
   - L2 场景总结（会话结束后生成 Markdown 摘要）
   - L3 用户画像（长期偏好聚合，跨会话复用）
5. **上下文卸载**：工具长输出（>1000 字符）自动转存外存，上下文保持精炼
6. **多轮对话**：前端 LocalStorage 持续保留，后端 SQLite 精确保存
7. **模型可替换**：改环境变量即可切换智谱/DeepSeek/混元/通义千问

## 界面要求

- 类似微信的聊天界面
- 用户消息在右边，AI 消息在左边
- 底部输入框 + 发送按钮
- 流式输出逐字显示
- 工具调用时显示"正在调用 xxx 工具..."提示
- 支持 Markdown 渲染 + 代码语法高亮

## 非功能需求

- API Key 仅存后端 `.env`，前端不泄露
- 后端 Python FastAPI，前端 React + Vite 分别启动
- 对话历史前端 LocalStorage + 后端 SQLite 双重存储
